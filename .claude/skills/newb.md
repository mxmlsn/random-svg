# Create New Branch

Create a new git branch and switch to it.

## Instructions

1. If the user provided a branch name as an argument, use it
2. If no branch name was provided, ask the user for the branch name
3. Run: `git checkout -b <branch-name>`
4. Confirm the branch was created and switched to by showing the output
